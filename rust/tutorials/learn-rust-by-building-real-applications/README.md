### What is Rust?

* Blazingly fast
* Memory Efficient
* No Runtime
* No Garbage Collector

* Rich type system
* Systems Programming Language
* Low level language
* Memory Safe (Code will not compile if memory issue)
* Thread Safe

* No Null (No nullptr)
* No Exceptions
* Rich Type System
* Modern Package Manager (Cargo)
* No data Races
